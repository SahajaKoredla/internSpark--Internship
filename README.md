# Diabetes Prediction Using Machine Learning

## Project Overview
This project predicts whether a person is diabetic or non-diabetic using Machine Learning.

## Dataset
The dataset used is the Pima Indians Diabetes Dataset.

## Algorithm Used
- Logistic Regression

## Libraries Used
- Pandas
- Scikit-learn

## Steps Performed
1. Loaded the diabetes dataset
2. Separated features and target variable
3. Split the dataset into training and testing data
4. Trained a Logistic Regression model
5. Evaluated the model using accuracy, confusion matrix, and classification report
6. Predicted diabetes status for a sample patient

## Model Accuracy
The model achieved an accuracy of 74.86%.

## Output
The model provides:
- Accuracy score
- Confusion matrix
- Classification report
- Diabetes prediction result

## How to Run

```bash
pip install pandas scikit-learn matplotlib seaborn
```

2. Run the Python file:
